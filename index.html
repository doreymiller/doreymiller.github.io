I recently built Ekko , an open source framework that provides realtime infrastructure and in-transit message processing for web applications.

Ekko is an open source framework that provides realtime infrastructure and in-transit message processing for web applications.

It allows for web developers to easily add realtime features to their web apps by abstracting away the complexity of managing realtime connections, which can become challenging when these features and connections need to scale.

Ekko also provides the ability to add serverless functions as custom middleware, allowing developers to apply business logic to their realtime communications in a modular and reusable way.
